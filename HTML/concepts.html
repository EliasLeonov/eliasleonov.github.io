Por que no es recomendable usar el tag 'br'

Definiciones

  • URI

    • Una URI es una cadena de caracteres que identifica un recurso abstracto o físico. Se puede subdividir en: Uniform Resource Locator (URL) y Uniform Resource Name (URN).

  • URL

    • URL son las siglas en inglés de Uniform Resource Locator. Como tal, el URL es la dirección específica que se asigna a cada uno de los recursos disponibles en la red con la finalidad de que estos puedan ser localizados o identificados. Así, hay un URL para cada uno de los recursos (páginas, sitios, documentos, archivos, carpetas)

  • HTML

    • HTML significa Hyper Text Markup Language. HTML es un caso particular de SGML (Standard Generalized Markup Language). Un archivo HTML es un archivo de texto que contiene markup tags. Cada tag HTML describe cierta parde del documento

  • XHTML

    • EXtensible HyperText Markup Language. Pensado para reemplazar al HTML. Es casi idéntico al HTML 4.01. Es una versión más estricta y limpia del HTML. Es HTML definido como una aplicación XML. Era la recomendación de la W3C hasta "!DOCTYPE html"

  • CSS

    • Cascading Style Sheets. Trabaja con HTML y otros lenguajes de markup controlando la manera en que se presenta el contenido.
